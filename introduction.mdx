## Quick Start

Get your first documentation page live in minutes. Follow these steps to set up and publish.

<Steps>
  <Step title="Create a New Page" icon="plus">
    Navigate to the dashboard and click "New Page". Choose a template like "API Reference" or "Guide".
  </Step>
  <Step title="Add Content" icon="edit-3">
    Use the MDX editor to write markdown, add components, and embed code examples.
    
````markdown
```javascript
// Example: Embed a simple API call
const response = await fetch('https://api.example.com/docs', {
  headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
});
```
````
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview changes, then publish. Share the live URL: `https://docs.example.com/your-project`.
  </Step>
</Steps>

## Multi-Language Support

Support global teams by adding translations or code examples in multiple languages.

<Tabs>
  <Tab title="JavaScript" icon="code">
    Initialize your docs client.
    
````javascript
const docsClient = new DocsClient({
  apiKey: 'YOUR_API_KEY',
  baseUrl: 'https://api.example.com'
});

await docsClient.pages.create({ title: 'My First Doc' });
````
  </Tab>
  <Tab title="Python" icon="python">
    Use the Python SDK for automation.
    
````python
from docs_client import DocsClient

client = DocsClient(api_key="YOUR_API_KEY")
client.pages.create(title="My First Doc")
````
  </Tab>
</Tabs>
